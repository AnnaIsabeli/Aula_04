# Sistema de Cadastro de Alunos

Projeto desenvolvido com Spring Boot e Thymeleaf para realizar o CRUD de alunos.

## Tecnologias utilizadas

* Java
* Spring Boot
* Thymeleaf
* HTML

## Como executar o projeto

1. Clonar o repositório

git clone https://github.com/seunome/sistema-alunos

2. Abrir o projeto em uma IDE (IntelliJ ou Eclipse)

3. Executar a classe principal do Spring Boot

4. Acessar no navegador

http://localhost:8080/alunos

## Funcionalidades

* Listar alunos
* Cadastrar aluno
* Editar aluno
* Excluir aluno
